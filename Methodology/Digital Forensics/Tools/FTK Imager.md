# FTK Imager

## Image and File System Acquisition

### 1) Load the disk image to FTK Imager

Live Acquisition

    Start FTK Imager -> File -> Add Evidence Item -> Logical Drive -> Select the OS Drive

Cold Acquisition

    Start FTK Imager -> File -> Add Evidence Item -> Image File -> Browse to the image file

## Registry data Acquisition

Live Acquisition

    Click Obtain Protected Files (Does not acquire all registry hives)

Live/Cold Acquisition

    Navigate to the location of the hives -> Select them -> Export Files to a location of our choice
