# OneDrive

Tools:

1) OneDriveExplorer (Run as admin)

Location:

    C:\Users\USER\Appdata\Local\Microsoft\OneDrive\logs

Enterprise environment important folder:

    C:\Users\USER\Appdata\Local\Microsoft\OneDrive\logs\Business

#### Important files within business folder

1) SyncEngine.odl

Contains all operations performed

2) SyncDiagnostics.log

Log file that contains diagnostic information related to the synchronization process of OneDrive

### 1) Metadata Harveting

In the OneDriveExplorer app, go to:

    File -> Live System

Then, in the leftside pane, browse for important files and directories and start analyzing.
